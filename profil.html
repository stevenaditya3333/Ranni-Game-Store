Profil Aplikasi

Deskripsi

Ranni Game Store adalah website toko game digital dan fisik yang dibuat untuk memenuhi tugas Pemrograman Web I. Aplikasi ini memungkinkan pengguna untuk:

  • Melihat daftar game unggulan dan terpopuler.
  • Menonton trailer dan media game.
  • Melihat harga game dengan jelas.
  • Melakukan transaksi dan pemesanan cepat secara online.
  • Mengelola akun profil (untuk fitur login sederhana).
  • Halaman beranda adalah halaman untuk serba instan
  • selalu klik Menu & petunjuk terlebih dahulu

Tentang Pembuat

Dibuat oleh Steven Aditya Pernando sebagai proyek pembelajaran dalam pengembangan web menggunakan HTML, CSS, dan JavaScript. Website ini menekankan tampilan gelap modern, navigasi intuitif, dan fitur interaktif.

Catatan

Semua transaksi dan demo pembayaran bersifat simulasi. Data transaksi disimpan secara lokal di browser menggunakan LocalStorage.
